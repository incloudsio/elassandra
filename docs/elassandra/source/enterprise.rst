Enterprise
==========

The maintained Elassandra.org repository and documentation currently focus on the
community distribution: **Apache Cassandra 4.0.x** with embedded **OpenSearch 1.3.x**.

No separate enterprise distribution is documented in this repository at this time.
If you are operating the current Elassandra line, use the community packaging,
configuration, and operational guidance from the rest of this documentation set.

For security, monitoring, and deployment hardening, start with:

* :doc:`configuration`
* :doc:`operations`
* :doc:`integration`
* the maintained Helm chart in `incloudsio/helm-charts <https://github.com/incloudsio/helm-charts>`_
